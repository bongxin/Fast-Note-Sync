#### 【活动者】
品控部

#### 【需求描述】

在 Web 端查看留仓申请全流程记录，支持按日期、流程状态、档口等条件筛选查询，查看申请、审核及确认详情，便于管理人员回溯与核查。

#### 【业务规则】

##### 标题
留仓记录

##### 筛选条件

|序号|字段名称|数据类型|必填|默认值|说明/交互规范|
|:--|:--|:--|:--|:--|:--|
|1|**日期范围**|日期范围选择器|否|近 7 天|选填。  <br>支持选择起止日期，格式 YYYY-MM-DD。  <br>默认展示近 7 天内的记录。|
|2|**流程状态**|下拉选择框|否|全部|**选项如下：**  <br>- 全部<br>- 待审核<br>- 待确认<br>- 已确认|
|3|**档口**|下拉选择框|否|全部|选填。  <br>选项取自系统档口主数据。|
|4|**申请人**|文本输入框|否|无|选填。  <br>支持按档口档主姓名模糊查询。|

##### 列表字段

Web 端以表格形式展示，支持分页。

|序号|字段名称|数据类型|说明/交互规范|
|:--|:--|:--|:--|
|1|**日期**|日期|展示申请日期，格式 YYYY-MM-DD。|
|2|**档口**|文本|展示申请档口。|
|3|**留仓申请原因**|文本|有内容时展示摘要（单行截断，超出显示省略号）；无内容时显示“—”。|
|4|**留仓头数**|数字|展示审核后的留仓头数；未审核时展示申请头数。|
|5|**申请人**|文本|展示档口档主姓名。|
|6|**流程状态**|标签|展示“待审核”“待确认”“已确认”。  <br>“待审核”“待确认”以醒目颜色标识。|
|7|**提交时间**|日期时间|展示申请提交时间，格式 YYYY-MM-DD HH:mm。|
|8|**审核人**|文本|已审核及之后状态展示审核人姓名；待审核时显示“—”。|
|9|**审核时间**|日期时间|已审核及之后状态展示审核时间，格式 YYYY-MM-DD HH:mm；待审核时显示“—”。|
|10|**官方兽医**|文本|已确认状态展示官方兽医姓名；未确认时显示“—”。|
|11|**确认时间**|日期时间|已确认状态展示确认时间，格式 YYYY-MM-DD HH:mm；未确认时显示“—”。|
|12|**操作**|操作列|显示“查看”链接。|

_列表默认按提交时间倒序排列。_

##### 详情展示

点击“查看”，在右侧抽屉（或弹窗）中展示该条留仓记录完整信息：

**申请信息**

|序号|字段名称|说明/交互规范|
|:--|:--|:--|
|1|**日期**|展示申请日期。|
|2|**档口**|展示申请档口。|
|3|**留仓申请原因**|完整展示文本内容。|
|4|**留仓头数**|展示审核后的留仓头数；未审核时展示申请头数。|
|5|**申请人签名**|展示申请人签名图片。|
|6|**申请人**|展示档口档主姓名。|
|7|**提交时间**|展示申请提交时间。|

**审核信息**（已审核及之后状态展示）

|序号|字段名称|说明/交互规范|
|:--|:--|:--|
|1|**审核人**|展示品控部审核人姓名。|
|2|**审核人签名**|展示审核人签名图片。|
|3|**审核时间**|展示审核完成时间。|

**确认信息**（已确认状态展示）

|序号|字段名称|说明/交互规范|
|:--|:--|:--|
|1|**官方兽医**|展示官方兽医姓名。|
|2|**官方兽医师签名**|展示官方兽医师签名图片。|
|3|**确认时间**|展示确认完成时间。|

_详情页为只读，不可编辑。_

##### 按钮

|序号|按钮名称|说明/交互规范|
|:--|:--|:--|
|1|**查询**|位于筛选区，点击后按当前筛选条件刷新列表。|
|2|**重置**|位于筛选区，点击后恢复筛选条件为默认值并刷新列表。|

#### 【操作流程】

1. **进入页面**：管理人员登录 Web 管理系统，进入【屠宰监管】-【留仓记录】，系统默认加载近 7 天内的留仓记录。
2. **筛选查询**：按需设置日期范围、流程状态、档口、申请人，点击“查询”刷新列表；点击“重置”恢复默认筛选。
3. **浏览列表**：在表格中查看各条记录的申请摘要、流程状态及审核、确认信息。
4. **查看详情**：点击某条记录的“查看”，在抽屉中浏览完整申请、审核及确认信息（按当前流程进度展示对应区块）。
