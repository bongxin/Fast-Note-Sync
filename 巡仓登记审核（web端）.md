#### 【活动者】
品控部

#### 【需求描述】

在 Web 端对小程序提交的巡仓登记记录进行审核，支持按条件筛选待审及已审数据，查看登记详情与佐证材料，并完成通过或驳回操作，形成闭环管理。

#### 【业务规则】

##### 标题
巡仓登记审核

##### 筛选条件

|序号|字段名称|数据类型|必填|默认值|说明/交互规范|
|:--|:--|:--|:--|:--|:--|
|1|**日期范围**|日期范围选择器|否|近 7 天|选填。  <br>支持选择起止日期，格式 YYYY-MM-DD。  <br>默认展示近 7 天内的记录。|
|2|**巡查情况**|下拉选择框|否|全部|**选项如下：**  <br>- 全部<br>- 正常<br>- 异常|
|3|**时间段**|下拉选择框|否|全部|**选项如下：**  <br>- 全部<br>- 08:00~10:00<br>- 10:00~12:00<br>- 12:00~14:00<br>- 13:00~16:00<br>- 16:00~18:00|
|4|**审核状态**|下拉选择框|否|待审核|**选项如下：**  <br>- 全部<br>- 待审核<br>- 已通过<br>- 已驳回|
|5|**巡查员**|文本输入框|否|无|选填。  <br>支持按巡查员姓名模糊查询。|

##### 列表字段

Web 端以表格形式展示，支持分页。

|序号|字段名称|数据类型|说明/交互规范|
|:--|:--|:--|:--|
|1|**日期**|日期|展示登记日期，格式 YYYY-MM-DD。|
|2|**时间段**|文本|展示所选巡查时间段。|
|3|**巡查情况**|标签|展示“正常”或“异常”。  <br>“异常”以醒目颜色（如红色）标识。|
|4|**巡查员**|文本|展示提交人姓名。|
|5|**提交时间**|日期时间|展示记录提交时间，格式 YYYY-MM-DD HH:mm。|
|6|**审核状态**|标签|展示“待审核”“已通过”“已驳回”。  <br>“待审核”以醒目颜色标识。|
|7|**审核人**|文本|已审核时展示审核人姓名；待审核时显示“—”。|
|8|**审核时间**|日期时间|已审核时展示审核时间，格式 YYYY-MM-DD HH:mm；待审核时显示“—”。|
|9|**操作**|操作列|待审核记录显示“审核”链接；已审核记录显示“查看”链接。|

_列表默认按提交时间倒序排列。_

##### 详情展示

点击“审核”或“查看”，在右侧抽屉（或弹窗）中展示该条巡仓登记的完整信息：

**登记信息**

|序号|字段名称|说明/交互规范|
|:--|:--|:--|
|1|**日期**|同登记页。|
|2|**时间段**|同登记页。|
|3|**巡查情况**|同登记页。|
|4|**存在问题**|完整展示文本内容；无内容时显示“—”。|
|5|**现场照片**|以缩略图展示；点击可放大预览，支持切换多张。  <br>无照片时显示“—”。|
|6|**巡查员签名**|展示签名图片。|
|7|**巡查员**|展示提交人姓名。|
|8|**提交时间**|展示记录提交时间。|

**审核信息**（已审核记录展示；待审核记录仅展示审核操作区）

|序号|字段名称|说明/交互规范|
|:--|:--|:--|
|1|**审核状态**|展示当前审核状态。|
|2|**审核人**|展示审核操作人姓名。|
|3|**审核时间**|展示审核完成时间。|
|4|**审核意见**|展示审核备注；无内容时显示“—”。|

##### 审核项

对待审核记录，审核人在详情抽屉底部填写并提交：

|序号|字段名称|数据类型|必填|默认值|说明/交互规范|
|:--|:--|:--|:--|:--|:--|
|1|**审核意见**|多行文本|否|无|选填。  <br>通过时可不填；驳回时**必填**。  <br>占位符提示：“请输入审核意见”。|

##### 按钮

|序号|按钮名称|说明/交互规范|
|:--|:--|:--|
|1|**查询**|位于筛选区，点击后按当前筛选条件刷新列表。|
|2|**重置**|位于筛选区，点击后恢复筛选条件为默认值并刷新列表。|
|3|**通过**|位于详情抽屉底部，仅待审核记录可见。  <br>点击后校验审核意见（通过时不强制），确认后将记录置为“已通过”。|
|4|**驳回**|位于详情抽屉底部，仅待审核记录可见。  <br>点击时校验审核意见必填，确认后将记录置为“已驳回”。|

#### 【操作流程】

1. **进入页面**：审核人员登录 Web 管理系统，进入【品控管理】-【巡仓登记审核】，系统默认加载近 7 天内、状态为“待审核”的记录列表。
2. **筛选查询**：按需设置日期范围、巡查情况、时间段、审核状态、巡查员，点击“查询”刷新列表；点击“重置”恢复默认筛选。
3. **浏览列表**：在表格中查看各条记录的登记摘要及审核状态；优先关注“待审核”及“异常”记录。
4. **审核处理**：点击待审核记录的“审核”，在抽屉中核对登记信息、存在问题及现场照片、签名；填写审核意见（驳回时必填），点击“通过”或“驳回”完成审核。
5. **查看已审记录**：点击已审核记录的“查看”，在抽屉中浏览登记信息与审核结果，不可再次修改审核结论。
